C#链表

节点定义

    public class Node<T>
 //C#中的internal访问修饰符表示 访问仅限于当前程序集
    {
        private T data;//数据
        private Node<T> next;
        //省略各种重载构造
    }

节点定义:

  • 值变量
  • 下一个对象变量
  • 构造器

赋值特性

    Node<int> n1 = new Node<int>();
    Node<int> n2 = new Node<int>(2);
    Node<int> n3 = new Node<int>(3);
    n1 = n2;
    n2 = n3;//n1.data=2
    n2.Data = 3;//n1.data=3

区别在于：

  • 对象赋值对象修改的是[变量→地址]，因此n1=n2不会影响到n1
  • 但是直接修改n2.Data等于只修改了[地址→值]，所以n1会受影响
    在后面的写法中要用到

思路/特性

C#的对象结构中也存在指针类型，但是更多的用类的思想来解决指针问题，

例如
[对象.变量=对象]的操作：

p=p.next;//对象=对象，其中next成员的定义是node对象

可以替代指针。

p=p->next;//对象->地址

应用思路

由上面两种特性，可以完成一些基本操作。
递归：

p=p.next;

替换: 

p1=p2;
p2=p2.next;//p1替换p2

插入：
p1.next=p2.next
p2.next=p1

断开链表：
ListNode new =p1.next;
p1.next=null;
return new;

链表实现完整代码

    public class LinkList<T> : IListDS<T>
    {
        private Node<T> head;
        public Node<T> Head
        {
            get { return head; }
            set { head = value; }
        }
        public T this[int index] => throw new NotImplementedException();

        public LinkList()
        {
            head = null;
        }//构造器初始化
        public int GetLength()
        {
            Node<T> p = head;
            int len = 0;
            while (p != null)
            {
                ++len;
                p = p.Next;
            }
            return len;
        }
        public void Clear()
        {
            head = null;
        }
        public bool IsEmpty()
        {
            if (head == null) return true;
            else return false;
        }
        //添加数据
        public void Add(T item)
        {
            Node<T> init = new Node<T>();
            Node<T> val = new Node<T>(item);
            if (head == null)
            {
                head = val;
                return;
            }
            while (head.Next != null)
            {
                head = head.Next;
            }
            head.Next = val;

        }
        //在单链表的第i个结点的位置前插入一个值为item的结点
        public void Insert(T item, int i)
        {
            if (IsEmpty() || i < 1)
            {
                Console.WriteLine("List is empty or Position is error!");
                return;
            }
            if (i == 1)
            {
                Node<T> q = new Node<T>(item);
                q.Next = head;
                head = q;
                return;
            }
            Node<T> p = head;
            Node<T> r = new Node<T>();
            int j = 1;
            while (p.Next != null && j < i)
            {
                r = p;
                p = p.Next;
                ++j;
            }
            if (j == i)
            {
                Node<T> q = new Node<T>(item);
                q.Next = p;
                r.Next = q;
            }
        }

        public T Delete(int i)
        {
            if (IsEmpty() || i < 0)
            {
                Console.WriteLine("Link is empty or Position is error!");
                return default(T);
            }
            Node<T> q = new Node<T>();

            if (i == 1)
            {
                q = head;
                head = head.Next;
                return q.Data;
            }
            Node<T> p = head;
            int j = 1;
            while (p.Next != null && j < i)
            {
                ++j;
                q = p;
                p = p.Next;
            }
            if (j == i)
            {
                q.Next = p.Next;
                return p.Data;
            }
            else
            {
                Console.WriteLine("The ith node is not exist!");
                return default(T);
            }
        }

        //获得单链表的第i个数据元素
        public T GetElem(int i)
        {
            if (IsEmpty())
            {
                Console.WriteLine("List is empty!");
                return default(T);
            }
            Node<T> p = new Node<T>();
            p = head;
            int j = 1;
            while (p.Next != null && j < i)
            {
                ++j;
                p = p.Next;
            }
            if (j == i)
            {
                return p.Data;
            }
            else
            {
                Console.WriteLine("The ith node is not exist!");
                return default(T);
            }
        }

        //在单链表中查找值为value的结点
        public int Locate(T value)
        {
            if (IsEmpty())
            {
                Console.WriteLine("List is Empty!");
                return -1;
            }
            Node<T> p = new Node<T>();
            p = head;
            int i = 1;
            while (!p.Data.Equals(value) && p.Next != null)
            {
                p = p.Next;
                ++i;
            }
            return i;
        }

    }

旋转链表

链接：https://leetcode.cn/problems/rotate-list/
给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。

实现思路

一开始想用滑动窗口的思想，把链表保存到数组，然后[数组]+[数组]拼接，滑动窗口求解。
但是那样做没有利用好链表的特性。

答案的思想：

  • 求出链表长度
  • 让链表头尾相连闭环
  • 在原链表的第 (n - 1) - (k \bmod n)(n−1)−(kmodn) 个节点断开链表即可

C#实现

    public ListNode RotateRight(ListNode head, int k) {
        ListNode L1=new ListNode();
        int count=1;
        L1=head;
        if(head==null||head.next==null)return head;
        if(k==0)return head;

        while(L1.next!=null){
            L1=L1.next;
            count++;
        }

        int pos=count-k%count;
        if(pos==count)return head;
        L1.next=head;

        while(pos>0){
            pos--;
            L1=L1.next;
        }
        ListNode L3=L1.next;
        L1.next=null;

        return L3;
    }
